Users

Add, remove, and manage users, link them to their connected service accounts, and control per-user sync.

User management lets you add and remove users, monitor license consumption, link each user to their accounts on integrated services, and monitor and control user-level sync.

The users list can be filtered by name and email address, and sorted by name or created date. Click a user's name or row to view and update details.

Editing users

In the user editing form, you can update:

  • The name displayed in user management
  • This user's Salesforce, MS365, Google, and/or Zoom user accounts
  • This user's email addresses

Nektar uses these linked external user accounts and email addresses to determine which user participated in a particular external communication.

Connectors

You can add the email address of a user under each connector you wish Nektar to read data from.

Sync toggles

These toggles let you control what data Nektar should sync for a user under a specific connector.

Admin

This toggle lets you add and remove Nektar administrators. Nektar administrators have full access to the console — they can add or remove users, manage connectors, and start or stop sync. All admins can add or remove other admins. Admins must have a Salesforce account.

Integration icons

These icons indicate a user's status within each connector. For connectors without user-level sync (such as Salesforce), a check mark is shown if the user is linked to a user account on that service.

For connectors with user-level sync (Google, Microsoft, Zoom), a sync health icon is shown for every object (emails, events). Click the icon for more information and to toggle sync on or off. Icons indicate the following states:

  • Disabled — can be enabled on the same screen
  • Not started — disabled globally, or pending approval
  • In progress — actively syncing; some data is remaining
  • Completed — everything synced; watching for new data
  • Delayed — sync has not run in the last 30 minutes
  • Stalled — stopped due to error or hasn't run in 2 hours

Add a user

There are two ways to add users — individually or in bulk.

Add individually

Click Add New User to open the user setup form. In the form:

  1. Select the user from the dropdown under each connector (Google, Microsoft, Zoom, etc.).
  2. Enable or disable the relevant syncs for that user.
  3. Turn on the Admin toggle to grant admin privileges if needed.
  4. Click Save. The new user will appear in your user list.

Import in bulk

Click Import Users to upload a CSV file. Download the CSV template and fill in:

  • User name
  • Email addresses for each connector you want to enable
  • Whether the user should have admin access

Upload the completed file. If there are errors, a pop-up will highlight them so you can correct and re-upload. Once successful, new users appear under the Users page.

Export Users

Click Export Users to download a CSV of all users, including connector information, sync preferences, and admin access levels.

Bulk actions

Select one or more users with the checkboxes at the start of each row to display bulk action buttons — enabling or disabling a particular sync, or deleting the selected users.
