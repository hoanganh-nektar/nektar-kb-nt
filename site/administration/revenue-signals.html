Revenue signals

Configure auto-updating Salesforce fields using Nektar's signal templates or custom rules.

Revenue signals let you auto-update fields on your Salesforce. You can place a request to set up a signal for any Salesforce object — Opportunity, Contact, Event, Task, Lead, Account, or Opportunity Contact Role.

When you select an object, you see the list of currently available signals with their status, description, and created date.

Request a new signal

Click Request for new signals to add more. You can choose from available templates or click Start from scratch.

Once requested, a support ticket is raised and the Nektar customer team works on your request, confirms the desired outcome, and publishes it to start auto-populating fields on your Salesforce.

Signal from template

You can request to add a template as-is or request edits to it. Editing an existing template does not change the template — it creates a new signal request from it.

Signal from scratch

  1. Choose either an existing Salesforce field or create a new field.
  2. Field type will be automatically populated if you select an existing Salesforce field.
  3. Define your rule in simple sentences. Include as many details as you wish.
  4. Submit request.

Track your pending requests on the signal list view.

Bulk actions

Select one or more signals with the checkboxes at the start of each row to display bulk action buttons — publish, un-publish, or delete.
