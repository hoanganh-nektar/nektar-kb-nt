Google Workspace

Connect Gmail and Google Calendar so Nektar can automatically capture customer interactions and write structured engagement data to Salesforce — with no rep involvement.

Connection

Your Google Workspace administrator should install Nektar from Google Marketplace.

Admin guide for how to install Nektar from Marketplace

Once the installation is completed, type your Google Workspace administrator's email address and click "Connect" to enable the connection within Nektar.

OAuth scopes

This is under-the-hood technical detail that's not necessary for using Nektar.

Scopes control what actions Nektar may take. Nektar requires the following OAuth scopes to operate correctly.

Nektar will only read Gmail messages, Google Calendar events and Google Meet meeting information (including transcripts) of users you select.

Sync

Nektar's Google Workspace sync system reads Gmail messages, Google Calendar events and Google Meet meeting information for the users for whom this has been enabled.

  • Gmail messages and Google Calendar events are read every 10–15 minutes.
  • Google Meet meeting information and transcripts are read every 10–15 minutes.
    • Please note, when first enabled, Google Meet can read data only for the last 30 days.
  • The list of users on your workspace is read every 24 hours.

You must enable sync for individual Google users in the Users section.

The sync table displays:

  • the time the most recent message or event was synced
  • a sync health icon that can be clicked for more information
  • toggles to enable or disable sync globally

Sync health icons may indicate the following states:

Disabled
Can be enabled on the same screen
Not started
Disabled globally, or pending approval
In progress
Actively syncing; some data is remaining
Completed
Everything synced; watching for new data
Delayed
Sync has not run in the last 30 minutes
Stalled
Stopped due to error or hasn't run in 2 hours
